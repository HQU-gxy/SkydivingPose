# 项目名称  

伞兵跳伞姿态估计。  

## 文件夹结构  

这个项目需要新建以下几个文件夹，每个文件夹的用途如下：  

1. `output`: 存放项目的输出文件，如运行结果等。  
2. `demo`: 存放演示所需的文件。  
3. `dataset`: 存放数据集文件。  
4. `checkpoint`: 存放模型的文件。  
5. `sample`: 存放样例数据或文件。  

checkpoint的模型文件可以从下载
